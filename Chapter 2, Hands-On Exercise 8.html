Three different Network Protocols

File Transfer Protocol (FTP)
FTP is a set of rules that allows files to be exchanged between computers on the internet

Hypertext Transfer Protocol (HTTP)
HTTP is a set of rules for exchanging files such as text, images, audio, video, and other multimedia on the Web

Transmission Control Protocol/Internet Protocol (TCP/IP)
The purpose of TCP is to ensure the integrity of network communication. It breaks down information into packets and then with IP that information is sent.
